CS 350 Embedded Systems work in Python at SNHU. Showcases my ability to work with embedded systems.

Projects entailed building towards programming a thermostat that was able to display when it was cooling, when it was heating, and then the ability to turn it off/swap between the three modes. 
Through the class, I was able to learn valuable embedded systems skills, such as how to work in a constrained environment, and also how solid state systems work.
